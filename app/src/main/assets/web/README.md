# TaskX (Adaptive Android UI)

Premium dark-theme Android-style schedule planner with responsive full-screen layout.

## Included UI

- No phone-frame shell; app fills available screen
- Material top app bar with menu, title, and online avatar
- Adaptive left drawer behavior by screen size:
- `Compact` (<700px): drawer closed on launch, overlay mode
- `Medium` (700px-1099px): drawer open on launch, overlay mode
- `Expanded` (>=1100px): persistent drawer pinned on left
- Today view with greeting, date, 68% progress ring, and schedule cards
- Bottom navigation on compact/medium screens
- Fully functional profile panel:
- Name, email, phone, date of birth, city, address, bio
- App password lock with unlock screen
- Profile and task data persist in browser `localStorage`

## Run

Open `schedule-app/index.html` in a browser.
